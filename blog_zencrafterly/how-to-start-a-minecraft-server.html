ติดตั้ง Java ก่อนรันเซิร์ฟ

PaperMC ต้องใช้ Java (JDK) — เวอร์ชัน Java ขึ้นกับเวอร์ชัน Paper/Minecraft ที่คุณดาวน์โหลด ไม่ใช่ติด Java ตัวเดียวใช้ได้ทุกเวอร์ชัน ดูตารางด้านล่างหรือหน้า PaperMC Getting started (อัปเดตล่าสุดจากทางการ)

ดาวน์โหลด JDK ได้ที่ Adoptium — ติดตั้งแล้วเปิด Command Prompt พิมพ์ java -version ตรวจว่าเวอร์ชันตรงกับ Paper ที่เลือก

สิ่งที่ต้องเตรียมก่อนเปิดเซิร์ฟเวอร์ Minecraft

ก่อนลงมือเปิดเซิร์ฟ ตรวจสอบสเปคเครื่อง พื้นที่ว่าง และ เวอร์ชัน Java ให้ตรงกับ Paper — ยิ่งมีผู้เล่นมากหรือใช้ Plugin มาก ยิ่งต้องการ RAM และ CPU สูงขึ้น

เวอร์ชัน Java ที่แนะนำตาม Paper (อ้างอิง PaperMC Docs)

ตัวรันเซิร์ฟเวอร์ Minecraft Java Edition ที่นิยมใช้

Vanilla คือตัวเกมต้นฉบับจาก Mojang — ตั้งค่าน้อยแต่ปรับแต่งจำกัด Spigot/Paper รองรับ Plugin เหมาะเซิร์ฟชุมชน — บทความนี้ใช้ PaperMC เพราะเสถียรและลด Lag ได้ดีกว่า Spigot

Spigot / Bukkit

  • รองรับ Plugin ของ Bukkit — ไม่รองรับ Mod โดยตรง
  • หลายเซิร์ฟย้ายมาใช้ Paper แทนเพื่อประสิทธิภาพที่ดีกว่า

PaperMC (ที่ใช้ในคู่มือนี้)

  • Fork จาก Spigot — เน้นประสิทธิภาพและแก้บั๊ก
  • รองรับ Plugin ของ Bukkit/Spigot ได้ — ไม่รองรับ Mod โดยตรง

วิธีเปิดเซิร์ฟเวอร์ Minecraft ด้วย PaperMC (Windows)

ขั้นตอนที่ 1: เตรียมโฟลเดอร์และไฟล์ PaperMC

  • 1.1 สร้างโฟลเดอร์ใหม่บน Desktop หรือไดรฟ์ที่ต้องการ
  • 1.2 ดาวน์โหลด PaperMC จาก papermc.io — เลือกเวอร์ชัน Minecraft ให้ตรงกับไคลเอนต์ที่เล่น
  • 1.3 นำไฟล์ .jar ที่ดาวน์โหลดมาใส่ในโฟลเดอร์ แล้วเปลี่ยนชื่อเป็น server.jar

ขั้นตอนที่ 2: สร้างสคริปต์รันเซิร์ฟ (run.bat)

  • 2.1 เปิด Notepad แล้ววางโค้ดนี้: 
    java -Xms2G -Xmx2G -jar server.jar --nogui
pause

    ปรับ -Xms และ -Xmx ตาม RAM ที่จัดให้เซิร์ฟ (เครื่อง 8 GB มักใช้ 2G–4G)

  • 2.2 บันทึกเป็น run.bat (ชนิดไฟล์ All Files)
  • 2.3 ดับเบิลคลิก run.bat — ครั้งแรกอาจปิดทันทีเพราะยังไม่ยอมรับ EULA
  • 2.4 เปิด eula.txt แก้ eula=false เป็น eula=true แล้วบันทึก
  • 2.5 รัน run.bat อีกครั้ง — เมื่อโหลดโลกเสร็จ พิมพ์ stop ในคอนโซลเพื่อปิดชั่วคราวก่อนตั้งค่า

ขั้นตอนที่ 3: ตั้งค่าและเข้าเล่นเซิร์ฟเวอร์

  • 3.1 เปิด server.properties ด้วย Notepad
  • 3.2 ตั้งค่า online-mode:
    • true — เฉพาะบัญชี Premium (แนะนำถ้าเล่นกับเพื่อนที่มีไอดีแท้)
    • false — รองรับไอดีปลอม (Offline) แต่เสี่ยงด้านความปลอดภัย
    หมายเหตุด้านความปลอดภัย: การปิด online-mode ทำให้คนแอบอ้างชื่อผู้เล่นคนอื่นได้ — ใช้เฉพาะเซิร์ฟส่วนตัวกับเพื่อนที่ไว้ใจเท่านั้น
  • 3.3 บันทึกไฟล์ แล้วรัน run.bat อีกครั้ง
  • 3.4 ในเกมไป Multiplayer → Direct Connection → ใส่ 127.0.0.1 หรือ localhost
  • 3.5 เข้าเล่นได้แล้ว! เพื่อนนอกเครื่องยังเข้าไม่ได้จนกว่าจะตั้งค่าเชื่อมต่อภายนอก (เช่น ngrok)

คำถามที่พบบ่อย

เปิดเซิร์ฟ Minecraft ต้องใช้ RAM เท่าไหร่?

เล่นกับเพื่อนไม่เกิน 5 คน มักจัดสรรให้เซิร์ฟ 2 GB ใน run.bat ถ้ามี 5–10 คนหรือใช้ Plugin แนะนำ 4 GB ขึ้นไป

ต้องใช้ Java เวอร์ชันไหนกับ Paper?

ขึ้นกับเวอร์ชัน Paper ที่ดาวน์โหลด — ดูตาราง เวอร์ชัน Java ตาม Paper หรือ PaperMC Docs มือใหม่ปี 2026 ที่เลือก Paper 1.20–1.21.11 มักใช้ Java 21 ส่วน Paper 26.1+ ต้องใช้ Java 25

Paper กับ Spigot ต่างกันยังไง?

Paper พัฒนาต่อจาก Spigot เน้นประสิทธิภาพและแก้บั๊ก — Plugin ส่วนใหญ่ใช้ร่วมกันได้ มือใหม่แนะนำ Paper

ทำไมเพื่อนเข้าเซิร์ฟไม่ได้แม้ผมเข้า localhost ได้?

localhost ใช้ได้แค่บนเครื่องคุณ — เพื่อนนอกบ้านต้องใช้ IP สาธารณะหรือเครื่องมือเช่น ngrok

โปรโมทเซิร์ฟให้คนอื่นเจอได้ที่ไหน?

ลงทะเบียนเซิร์ฟฟรีได้ที่ รายการเซิร์ฟเวอร์ Zencrafterly เพื่อให้ผู้เล่นค้นหาและโหวต

ขั้นตอนต่อไป: ทำให้เพื่อนเข้าเล่นได้!
